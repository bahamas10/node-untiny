untiny
======

Untiny a URL

Installation
------------

First, install [Node.js][0].  Then:

    npm install -g untiny

Usage
-----

    $ untiny http://tinyurl.com/c8bpv6z
     -> http://tinyurl.com/c8bpv6z
     -> http://tinyurl.com/cst2eyp
    http://www.daveeddy.com
